# 🏟️ Turf Booking System

A full-stack booking application built with React, Node.js, and PostgreSQL.

## 🚀 Live Demo
- **Frontend:** [Link to your S3/Domain URL]
- **API Endpoint:** `http://your-elastic-ip:3001`

## 🛠️ Tech Stack
- **Frontend:** React (Vite), Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** PostgreSQL (AWS RDS)
- **Hosting:** AWS EC2, S3

## ⚙️ Setup & Installation
1. Clone the repo: `git clone https://github.com/user/repo.git`
2. Install dependencies: `npm install`
3. Create a `.env` file based on the instructions below.
4. Run locally: `npm run dev`

## 📡 Environment Variables
To run this project, you will need to add the following environment variables to your .env file:
`VITE_API_BASE_URL` = Your Backend IP/URL
